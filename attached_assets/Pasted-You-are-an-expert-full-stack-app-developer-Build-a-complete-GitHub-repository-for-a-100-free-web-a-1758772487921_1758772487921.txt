3. Backend (Python FastAPI)

File: backend/app.py.

Expose endpoints:

/generate-video → calls free Hugging Face model API.

/health → returns {status: "ok"}.

Use Stable Video Diffusion (img2vid) or AnimateDiff from Hugging Face Hub (free inference endpoints).

Ensure character consistency by reusing the same uploaded image embedding.

Add requirements.txt with:

uvicorn
requests
huggingface_hub
python-multipart
